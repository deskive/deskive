## Deskiveとは？

Deskiveは、リアルタイムコミュニケーション、プロジェクト管理、ビデオ会議、ファイル管理、AIツールを一つのセルフホスト可能なプラットフォームに統合したオープンソースのワークスペースです。

## 主な機能

- **リアルタイムチャット** — チャンネル、スレッド、リアクション、GIF対応
- **ビデオ会議** — LiveKit経由のHD通話、画面共有、録画、文字起こし
- **プロジェクト管理** — カンバン、スプリント、タスク、タイムトラッキング
- **ファイル管理** — クラウドストレージ、バージョン管理、共有
- **カレンダー** — イベント、会議室、空き状況管理
- **ノート** — リアルタイムコラボレーション対応ブロックエディタ
- **AI AutoPilot** — 日次ブリーフィング、タスク管理、会議インテリジェンス
- **ホワイトボード** — コラボレーティブビジュアルワークスペース

## クイックスタート

```bash
git clone https://github.com/deskive/deskive.git
cd deskive/backend && cp .env.example .env && npm install && npm run migrate && npm run start:dev
```
