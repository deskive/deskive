## Deskive란?

실시간 커뮤니케이션, 프로젝트 관리, 화상 회의, 파일 관리, AI 도구를 하나의 셀프 호스팅 가능한 플랫폼으로 통합한 오픈소스 워크스페이스입니다.

## 주요 기능

- **실시간 채팅** — 채널, 스레드, 리액션, GIF 지원
- **화상 회의** — LiveKit 기반 HD 통화, 화면 공유, 녹화, 자막
- **프로젝트 관리** — 칸반, 스프린트, 작업, 시간 추적
- **파일 관리** — 클라우드 스토리지, 버전 관리, 공유
- **캘린더** — 이벤트, 회의실, 가용성 관리
- **노트** — 실시간 협업 블록 에디터
- **AI AutoPilot** — 일일 브리핑, 작업 관리, 미팅 인텔리전스
