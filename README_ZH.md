## 什么是Deskive？

Deskive是一个开源工作空间协作平台，将实时通信、项目管理、视频会议、文件管理和AI工具整合到一个可自托管的应用中。

## 主要功能

- **实时聊天** — 频道、线程、表情回复、GIF支持
- **视频会议** — 通过LiveKit实现高清通话、屏幕共享、录制、转录
- **项目管理** — 看板、冲刺、任务、时间跟踪
- **文件管理** — 云存储、版本控制、共享
- **日历** — 事件调度、会议室、可用性管理
- **笔记** — 支持实时协作的块编辑器
- **AI AutoPilot** — 每日简报、任务管理、会议智能
- **白板** — 协作式视觉工作空间
