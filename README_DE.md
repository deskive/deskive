## Was ist Deskive?

Open-Source-Kollaborationsplattform, die Echtzeit-Kommunikation, Projektmanagement, Videokonferenzen, Dateiverwaltung und KI-Tools in einer selbst gehosteten Anwendung vereint.

## Hauptfunktionen

- **Echtzeit-Chat** — Kanaele, Threads, Reaktionen, GIFs
- **Videokonferenzen** — HD-Anrufe ueber LiveKit mit Aufzeichnung und Transkription
- **Projektmanagement** — Kanban, Sprints, Aufgaben, Zeiterfassung
- **Dateiverwaltung** — Cloud-Speicher, Versionierung, Teilen
- **Kalender** — Termine, Besprechungsraeume, Verfuegbarkeit
- **Notizen** — Block-Editor mit Echtzeit-Zusammenarbeit
- **KI AutoPilot** — Taegliche Briefings, Aufgabenverwaltung, Meeting-Intelligenz
