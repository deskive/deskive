## Que es Deskive?

Plataforma de colaboracion de codigo abierto que integra comunicacion en tiempo real, gestion de proyectos, videoconferencias, archivos y herramientas de IA en una aplicacion auto-hospedable.

## Funciones principales

- **Chat en tiempo real** — Canales, hilos, reacciones, GIFs
- **Videoconferencias** — Llamadas HD via LiveKit con grabacion y transcripcion
- **Gestion de proyectos** — Kanban, sprints, tareas, seguimiento de tiempo
- **Gestion de archivos** — Almacenamiento en la nube, versionado, compartir
- **Calendario** — Eventos, salas de reuniones, disponibilidad
- **Notas** — Editor de bloques con colaboracion en tiempo real
- **AI AutoPilot** — Briefings diarios, gestion de tareas, inteligencia de reuniones
