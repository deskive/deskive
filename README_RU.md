## Что такое Deskive?

Открытая платформа для совместной работы, объединяющая коммуникации в реальном времени, управление проектами, видеоконференции, файлы и ИИ-инструменты в одном приложении с возможностью самостоятельного размещения.

## Основные возможности

- **Чат в реальном времени** — Каналы, ветки, реакции, GIF
- **Видеоконференции** — HD-звонки через LiveKit с записью и транскрипцией
- **Управление проектами** — Канбан, спринты, задачи, учёт времени
- **Управление файлами** — Облачное хранилище, версионирование, общий доступ
- **Календарь** — События, переговорные, управление доступностью
- **Заметки** — Блочный редактор с совместной работой в реальном времени
- **ИИ AutoPilot** — Ежедневные сводки, управление задачами, аналитика встреч

Подробности в [английском README](./README.md).
